Tecnicas del Parkour

Domina los movimientos fundamentales y avanzados del arte del desplazamiento

Tecnicas Recomendadas para principiantes

Salto de Precision

Salto controlado hacia superficies pequeñas o estrechas, exigiendo alta precision al aterrrizar

Dificultad: Baja
Rodada (Roll)

Amortiguacion segura tras un salto, rodando diagonalmente sobre un hombro para dispersar la energia.

Dificultad: Baja
Safety Vault

Vault basico para superar obtaculos con fluidez, apoyando las manos para mayor seguridad.

Dificultad: Baja

Tecnicas Esenciales del Parkour

Roll (Rodada)

Tecnica para absorber impactos tras saltos altos, rodando diagonalmente sobre un hombro para dispersar la energia con fluidez.

Precision Jump

Salto controlado hacia superficies pequeñas o estrechas, exigiendo alta precision al aterrizar.

Arm Jump

Salto que culmina agarrandose al borde de un obstaculo con ambas manos, util para trepar.

Wall Run

Carrera hacia una pared, impulsandose con un pie para ganar altura o superar el obstaculo.

Climb Up

Pasar de estar colgado del border a situarse sobre el, tecnica fundamental para superar muros.

Cat Leap

Salto a una pared o cornisa, colgandose con las manos para trepar o estabilizarse.

Otras Tecnicas Destacadas

Pasavallas (Vaults)

Variados tipos como speed vault, lazy vault o dash vault, que permiten superar obstaculos apoyando las manos.

Tic Tac

Impulso lateral desde una pared para alcanzar una altura adicional o cambiar de direccion.

Recepciones y Aterrizajes

Incluyen recepcion Basic -flexionando rodilla y evitando caer de talones—, rodadas y recepciones adaptadas.

Equilibrio y Balanceo

Caminar por bordes estrechos o usar barras para balancearse y sortear obstaculos.

Tecnicas Combinadas Avanzadas

Wallflips

Combinacion de wall run con volteretas, demostrando dominio completo del movimiento vertical

Wallruns Combinados

Secuencias de wall runs que incluyen saltos, volteretas y desplazamiento por superficies verticales.

Secuencias Complejas

Combinaciones de multiples tecnicas en secuencia fluida, creando rutas de parkour dinamicas.

Consejos de Seguridad

Antes de entrenar
  • ✓ Calienta siempre antes de practicar
  • ✓ Verifica el terreno y obstaculos
  • ✓ Usa ropa y calzado apropiados
  • ✓ Entrena con un compañero
  • ✓ Comienza con tecnicas Basicas
Durante la practica
  • ✓ Escucha Tu cuerpo
  • ✓ Progresa gradualmente
  • ✓ Manten la concentración
  • ✓ Respeta tus limites
  • ✓ Practica en areas seguras
